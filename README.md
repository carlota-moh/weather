# Weather MCP

POC for an MCP server that retrieves weather data. Done using this [tutorial](https://modelcontextprotocol.io/quickstart/server#node).

## Running the server

1. Install dependencies:

   ```bash
   npm install
   ```

2. Build the project:

   ```bash
   npm run build
   ```

3. Start the server:

   ```bash
    npm start
   ```
